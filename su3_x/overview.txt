
filename format for data files: su3_x_{L}_{N5}_{beta}_{eps5}.db

L: lattice size
N5: number of slices in extra dimension
beta: lattice coupling * 1000
eps5: lattice coupling ratio for extra dimension * 1000 (for N5 != 1)
n: number of data points
n_sub: number of jackknife bins
sqrt(sigma): square root of string tension (using creutz ratios)
d_tau: hybrid monte carlo time per configuration
DQ: average change in normalized topological charge per configuration
t_DQ: average hybrid monte carlo time per change in topological charge (d_tau / DQ)
